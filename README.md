## 📌 About

Sky Runner is a simple 2D endless runner made in Unity.
The player must avoid incoming birds and survive as long as possible.

## ⚙️ Features

* Smooth player movement
* Enemy spawning system
* Collision detection
* Game Over system with restart button
* Sound effects

## 🛠 Tech

* Unity (2D)
* C#

## 🎯 Gameplay

The goal is to survive as long as possible while avoiding obstacles.
The game ends when the player collides with a bird.
